# STM32-learning

This repo contains my learning journey of the STM32 ecosystem, the basic communication protocols (UART, I2C, SPI) and RTOS. 

My goal is to tackle these goals separately and then in a final project combine my knowledge of the hardware and software. I will primarily code in bare-metal C/C++ but I will utilize the HAL of the STM32 by using the "STM32CubeMx" program.

## Hardware

- Nucleo-F411RE
- SSD1309 OLED Display
- BME280 Environmental Sensor
- GY-521 (MPU6050 IMU)
- GY-NEO8MV2 GPS Module

## Software & Tools

- STM32CubeIDE
- STM32CubeMX
- Logic-24M-8CH Logic Analyzer
