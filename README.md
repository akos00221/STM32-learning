# STM32-learning

This repo contains my learning journey of the STM32 ecosystem, the basic communication protocols (UART, I2C, SPI) and also get into RTOS. My goal is to tackle these goals separately and then in a final project combine my knowledge of the hardware and software. I will primarily code in bare-metal C/C++ but I will utilize the HAL of the STM32 by using the "STM32CubeMx" program.

## Hardware used:
Nucleo-F411Re
SSD1309
BME-280-M
GY-521
GY-NE08MV2

## Utilities:
STM32CubeIDE
STM32CubeMx
Logic-24M-8CH - Logic analyzer
